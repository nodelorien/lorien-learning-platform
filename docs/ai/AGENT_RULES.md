# AI Agent Rules

Never create endpoints without OpenAPI update.

Never modify ADRs.

Never access production credentials.

Always create tests.

Always update documentation.

Use Typescript strict mode.

Follow hexagonal architecture.

Follow S.O.L.I.D principles

Follow codemetrics big-o notation weights, no more 15 lines by function.

Follow clean code patterns and practices.

Follow functional approach for typescript.

Follow Dependency injection pattern / Control inversion dependency.

Follow injection by constructor and strictly by interface.

Follow components patterns with a manager facade to expose component functions.

All shared components must be abstract and must be injected by DI.
